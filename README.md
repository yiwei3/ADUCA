# Adaptive Delayed-Update Cyclic Algorithm

## Overview

This repository contains experiments code of the research project of Adaptive Delayed-Update Cyclic Algorithm ($\texttt{ADUCA}$). We implement the algorithm in the Support Vector Machine and linearly constrained Huber-Elastic minimization problems. We compare the performance with other baseline algorithms in these two problems.

## Running the experiments

Use ./SVM/main.py and ./lc_huber_elastic/main.py.
